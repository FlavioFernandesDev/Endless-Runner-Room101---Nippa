Descrição: Este é um jogo to tipo Endless Runner em 3D onde o jogador controla o porteiro de um Hotel que percorre infinitamente os corredores do edifício. O objetivo é percorrer a maior distância possível enquanto gere o tempo e a precisão dos movimentos para evitar colisões fatais.
